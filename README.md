# TowerDefenseGame
Solo TD

## Projektstruktur

- `Assets/` enthält die Unity-Inhalte des Spiels.
- `Assets/Scripts/` enthält die MonoBehaviour-Skripte und Datenklassen für Game-Loop, Bau-, Wave-, Gegner-, Tower- und Chaos/Gerechtigkeit-Systeme.
- `Assets/Prefabs/` enthält wiederverwendbare Unity-Prefabs für Tiles, Tower, Gegner und Projektile.
- `Assets/Resources/` enthält zur Laufzeit ladbare Assets, aktuell vor allem Tower-Icons.
- `Assets/Settings/` enthält Unity-/URP-Projekteinstellungen und Build-Profile.
- `Packages/` enthält Unity-Package-Manifest und Lockfile.
